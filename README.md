# Nuvora Digital Landing Page

Landing page statis bertema biru minimal elegan untuk digital product/jasa.

## Isi

- `index.html` — struktur halaman utama dengan menu Product
- `styles.css` — styling responsive tema biru
- `script.js` — menu mobile, header elevation, reveal animation
- `db.json` — database decoy berisi 5 segmen, 5 jenis, dan 5 produk

## Data Decoy

- 5 segmen: Startup Founders, Service Business, Personal Brand, Online Education, Local Business
- 5 jenis: Landing Page, Brand Kit, Automation Flow, Sales Funnel, Dashboard Lite
- 5 produk: Nuvora Launch One, Signal Kit, Flow Start, Course Funnel, Local Pro

## Jalankan Lokal

Buka `index.html` langsung di browser, atau pakai server lokal:

```bash
python3 -m http.server 8080
```

Lalu buka `http://localhost:8080`.
