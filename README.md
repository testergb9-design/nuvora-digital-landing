# Nuvora Digital Landing Page

Landing page statis minimal elegan untuk digital product/jasa.

## Isi

- `index.html` — struktur halaman utama
- `styles.css` — styling responsive minimal elegan
- `script.js` — menu mobile, header elevation, reveal animation

## Jalankan Lokal

Buka `index.html` langsung di browser, atau pakai server lokal:

```bash
python3 -m http.server 8080
```

Lalu buka `http://localhost:8080`.
